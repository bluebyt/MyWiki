
## Add rounded corners to an image or animated .webp in GIMP

1. Open image.webp.
(If already enabled, it will be greyed out — that’s fine)

2. Layer → Transparency → Add Alpha Channel

3. Select all frames.

4. Select the Rectangle Select Tool. (Bottom right side)

5. Enable Rounded corners and set the radius (e.g., 15).

6. Select All.

7. Click on the image (make sure the selection outline is not visible on the image).

8. Select Invert.

9. Press Delete.

10. Wait until all frames are processed (this can take about 15 seconds).

11. Go to File → Export As.

12. Enable Save as animation.

13. Enable Loop forever, then export.
